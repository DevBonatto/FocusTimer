Esse projeto foi desenvolvido com as seguintes tecnologias:

## 💻 Projeto

Neste desafio foi criado um Focus Timer parecido com o Pomodoro. Temas aplicados:

- Funções no Javascript
- Manipulação da DOM
- Funções
- Callback Functions
- ESModules
- Clean Code
- States
- Adicionando áudios
